Font Download Instructions

Important: To ensure proper Polish character support in PDF exports, please download the Roboto font and place it in this directory.

Follow these steps:

  1. Download Roboto Regular from Google Fonts
  2. Extract the ZIP file
  3. Find the file named Roboto-Regular.ttf
  4. Place that file in this directory (public/fonts/)

After adding the font file, your PDF exports will correctly display Polish characters.

The application looks for this file at: /fonts/Roboto-Regular.ttf
